# Orbit Wars AI Project

Kaggle Orbit Wars 대회 참가를 위한 AI 에이전트 개발 프로젝트입니다.

## 파일 구성
- `advanced_agent.py`: 최신 V3 전략이 반영된 메인 에이전트 (궤도 예측, 태양 회피, 효율적 행성 점령 로직 포함)
- `farmer_agent.py`: 고득점 파밍을 위해 의도적으로 적 행성 하나를 남겨두고 병력을 증식시키는 특수 에이전트
- `test_agent.py`: 전략 테스트 및 벤치마킹용 후보 에이전트
- `sim.py`: 로컬 환경에서 에이전트 간 대결을 시뮬레이션할 수 있는 스크립트
- `orbit_wars.py` / `orbit_wars.json`: 게임 엔진 및 환경 설정 파일

## 주요 성과
- **V3 전략**: 100% 병력 집중 및 최단 경로 궤도 계산을 통해 로컬 시뮬레이션에서 압도적인 승률 기록
- **대회 제출**: Kaggle 서버에 최종 에이전트 제출 완료
